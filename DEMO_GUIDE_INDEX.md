## 📞 Need Help During Demo?

**Quick troubleshooting:**
1. Check browser console for JavaScript errors
2. Check backend terminal for API errors
3. Make sure backend is running on port 6060
4. Make sure frontend is running on port 3000
5. Hard refresh (Ctrl+Shift+R) on browser
6. Check JWT token is not expired

---

## 🎓 Final Tips

1. **Practice once before demo** - Run through all 5 steps yourself
2. **Have backup screenshots** - Just in case something fails
3. **Know your numbers** - "Found 1 overdue prospect, sent 1 email"
4. **Be confident** - You built this, you know it well
5. **Go slow** - Better to pause and explain than rush
6. **Watch mentor's reactions** - Adjust pace if needed
7. **Ask if they have questions** - "Does that make sense?"
8. **End strong** - "Any other questions?" or "Want me to deploy?"
